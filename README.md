# 💳 Credit Card Finance Dashboard

## 📊 Overview

The Credit Card Finance Dashboard is a data visualization project designed to analyze and monitor credit card transactions, spending behavior, and financial trends. It provides actionable insights to help users understand expenses, manage budgets, and make informed financial decisions.

---

## 🚀 Features

* 📈 **Spending Analysis** – Track total spending across categories (e.g., groceries, travel, entertainment)
* 💰 **Income vs Expense** – Compare income and credit usage over time
* 🧾 **Transaction Insights** – View detailed transaction-level data
* 🏦 **Card Utilization** – Monitor credit limits and usage ratios
* 📅 **Time-based Trends** – Analyze monthly and yearly patterns
* 🔍 **Interactive Filters** – Drill down by category, date, or card type

---

## 📂 Dataset

The dataset includes:

* Transaction ID
* Date
* Amount
* Category (Food, Travel, Bills, etc.)
* Card Type
* Credit Limit
* Customer Information (if applicable)
